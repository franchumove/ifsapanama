BRIEF / SPEC PARA REPLIT AGENT — IFSA PANAMÁ (TRS-Funnel™ Website)
0) Objetivo del sitio (TRS)

Construir una web que no “muestre” servicios, sino que mueva estado interno del visitante:

Transformar (0–250ms): “Esto se ve serio. Control.”

Resonar (250ms–3s): “Me baja el riesgo. Confío.”

Simbolizar (3–10s): “Quiero una visita técnica / cotización.”

Acción primaria a optimizar

Solicitar visita técnica (CTA principal, siempre visible).

Acciones secundarias

Ver proyectos (filtro por tipo)

Contacto (WhatsApp + formulario)

Descargar perfil (si se crea PDF luego)

1) Branding & UI System
Paleta (base)

Negro: #0B0B0B (fondo hero, header, textos fuertes)

Dorado: #C9A227 (CTAs, iconos, acentos, números)

Blanco: #FFFFFF (fondos de lectura)

Gris claro: #F5F5F5 (secciones alternas)

Gris texto: #2B2B2B

Regla: Dorado solo para “señales” (botones, iconos, líneas, números). No usar dorado en párrafos largos.

Tipografía (Google Fonts)

Titulares: Montserrat o Poppins (700/800)

Cuerpo: Inter (400/500)

Estilo visual

Minimalista, industrial, “control de obra”.

Muchas fotos reales (obra, maquinaria, topografía, personal).

Cards con borde sutil y sombras suaves.

Animaciones discretas (fade/slide) sin exagerar.

2) Tech stack (implementación)

Construir con:

Next.js (App Router) + TypeScript

Tailwind CSS

Componentes: shadcn/ui (Button, Card, Tabs, Accordion, Dialog)

Animaciones: Framer Motion (suave)

SEO: metadata por página + OpenGraph

Form: validación con zod + react-hook-form

Envío de formulario: endpoint /api/lead (guardar a email + fallback a mailto si no hay SMTP)

Integración WhatsApp: link directo con mensaje prellenado

Responsive first (mobile decide).
